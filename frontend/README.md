# CVF PT frontend

React 19 and Vite 6 frontend for CVF PT. Create React App and `react-scripts` are
not used.

```sh
npm install
npm run dev
npm run build
npm run test:e2e:preview
```

The production build is written to `dist/`. See the repository `README.md` for
verification and `DEPLOYMENT.md` for the isolated Vercel frontend project setup.
